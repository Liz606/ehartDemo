## 技术背景

基于React+Redux+Webpack搭建的前端脚手架，开发者需要掌握es6、react、redux等相关知识，编译工具需要掌握Wepack相关知识。

## 开发环境

- 安装Homebrew，命令行输入

```Shell
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

- IDE：Atom、Sublime Text、VSCode等不限
